前言

本文属于黑金 AX7010/AX7020开发教程的第二篇文章，主要介绍基于 Zynq 7000 的 PL 部分实现的 “Hello World” 程序——LED实验，教程源自芯驿电子 ALINX：黑金 AX7010/7020 开发配套教程 ，本文及后续文章都是根据此教程撰写。

关键名词

PS（Processing System，处理系统）

就是与 FPGA 无关的 ARM 的 SoC 的部分，包含应用处理单元（Application Processing Unit，APU）、扩展外设接口（与 PL 外设不能直接互通，包含GPIO、以太网、USB、SCU、串口等）、存储器接口（DDR3）、互联接口、时钟发生电路、以及cache 存储器等电路组成

APU(应用处理器单元 ) 主要是由两个 ARM 处理核组成。

PL（Programming Logic，可编程逻辑）

就是FPGA部分。是由逻辑片和可配置逻辑块 （Configurable Logic Block，CLB）组成的，另外还有用于接口的输入 / 输出块（Input/Output Block，IOB)

ZYNQ，分为 PS 部分和 PL 部分，就是 ARM 的 SoC 部分和 FPGA 部分
